# netmusic

基于Vue、ElementUI、NodeJs的网易云音乐

## 使用

### 初始化
```shell
npm run init
```

### 本地服务
```shell
npm run start
```

### 开发服务
```shell
npm run dev
```

### 构建
```shell
npm run build
```

### 测试
```shell
npm run test
```

### 检测
```shell
npm run lint
```
